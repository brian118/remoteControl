# 一个基于Vue.js的Electron的CSR项目模板

## Installation & Quick start

### 安装

先切回到公司内部npm源。

Windows系统安装
```
$ npm install dc-cli -g
```

Mac下安装
```
$ sudo npm install dc-cli -g
```

### 查看帮助信息

```
$ dc
```


### 创建项目

```
# 指定项目名字创建项目
$ dc create 模板名<template-name> 项目名字<project-name>

# 在当前目录创建项目（!!!注意模板名后面加.）
$ dc create 模板名<template-name> .
```

### 查看所有支持的项目模板

```
$ dc list
```
